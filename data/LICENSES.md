# Data Source Licenses

The JSON output in /data redistributes information from the following sources.
Consumers MUST honor each source's license:

- TheSportsDB — Free for non-commercial use; attribution appreciated.
- Football-Data.org — Free tier, attribution required.
- OpenFootball — Public Domain (Unlicense / CC0).
- StatsBomb Open Data — CC BY-NC-SA 4.0, attribution to StatsBomb mandatory.
- Wikidata — CC0.
- OpenStreetMap — ODbL; "© OpenStreetMap contributors" required.
